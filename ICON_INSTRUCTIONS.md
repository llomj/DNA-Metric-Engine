# Icon Generation Instructions

To generate the cyberpunk-style PWA icons:

1. Open `public/create-icons.html` in your web browser
2. The page will automatically render the icons
3. Click the "Download 192x192" button to download the small icon
4. Click the "Download 512x512" button to download the large icon
5. Save both files as `pwa-192x192.png` and `pwa-512x512.png` in the `public/` folder

The icons feature:
- Black background (#000000)
- Emerald green DNA helix (#10b981)
- Cyberpunk aesthetic with grid pattern
- Tech border frame
- "DNA" text branding
- Glowing nodes and connections

These icons will be used when users add the app to their home screen.
